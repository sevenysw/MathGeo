Python Package Requirements:

Pytorch

scipy

h5py

Python File description:

train.py：train the POCS-Net.

prediction.py：interpolate test data using pre-trained POCS-Net.

loss_validation_computation.py: compute the loss of the dataset.

SNR_training_computation.py: compute SNR of the dataset.

The datasets and pretrained model links are listed as follows:
2D: https://www.kaggle.com/datasets/supercy8250/ssl-2d-pocs-net-public

3D: https://www.kaggle.com/datasets/supercy8250/ssl-3dpocs-net-public
